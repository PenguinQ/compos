# Icons
Icon component are used to render all icons used in the applications as a **web component** that render the icon as an inline SVG.

By default, icon size are set to **24px**.

## Usage
To start using the icons, import the component `ComposIcon` and the icon you needs, see the example below:

<Canvas of={IconsStories.DocUsage} />

## Properties
| Name  |  Type  | Default Value | Description                                                                                  |
| ----- | :----: | :-----------: | -------------------------------------------------------------------------------------------- |
| icon  | string |       -       | SVG string to be rendered, it can also render non-included imported icon provided by default |
| name  | sring  |       -       | Works the same as icon, for fallback purposes.                                               |
| size  | string |       -       | Set the size of the icon, accept any CSS sizing                                              |
| color | string |       -       | Set the color of the icon, accept any color format                                           |
