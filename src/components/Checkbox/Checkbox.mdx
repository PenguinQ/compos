# Checkbox
Checkbox is... well, a box... to be checked!

## Usage
To start using **Checkbox**, see the example below:

<div className="demo-block">
  <Story of={CheckboxStories.DocUsage} />

  ```tsx
  <script setup lang="ts">
  import { ref } from 'vue';

  import { Checkbox } from '@/components';

  const checked = ref(false);
  </script>

  <template>
    <Checkbox v-model="checked" label="Insight Mode" />
  </template>
  ```
</div>

## Examples
### Non Two-way Data Binding
The recommended way to use **Checkbox** is using Vue two-way data binding (v-model). But, you can also go the other route
using `value` and `@change`, see the example below:

<div className="demo-block">
  <Story of={CheckboxStories.DocUsingValue} />

  ```tsx
  <script setup lang="ts">
  import { ref } from 'vue';

  import { Checkbox } from '@/components';

  const checked = ref<boolean | string>(false);

  const handleChange = (e: Event) => {
    const target = e.target as HTMLInputElement;
    const targetChecked = target.checked;

    if (targetChecked) {
      checked.value = target.value;
    } else {
      checked.value = false;
    }
  };
  </script>

  <template>
    <Text>Simulated Universe Insight Mode: {{ checked }}</Text>
    <Checkbox :value="true" label="Insight Mode" @change="handleChange" />
  </template>
  ```
</div>

### Multiple Selection
You can also have multiple selection of Checkboxes in a single v-model value, just set the v-model value as an array and
set the every Checkboxes v-model with the same v-model, see the example below:

<div className="demo-block">
<Story of={CheckboxStories.DocMultiple} />

```tsx
<script setup lang="ts">
import { ref } from 'vue';

import { Checkbox } from '@components';

const selected = ref([]);
</script>

<template>
  <Text margin="0 0 16px">Selected Checkboxes: {{ selected }}</Text>
  <div style="display: flex; gap: 16px;">
    <Checkbox v-model="selected" value="Kafka" label="Kafka" />
    <Checkbox v-model="selected" value="Himeko" label="Himeko" />
    <Checkbox v-model="selected" value="Natasha" label="Natasha" />
    <Checkbox v-model="selected" value="Ruan Mei" label="Ruan Mei" />
  </div>
</template>
```
</div>
